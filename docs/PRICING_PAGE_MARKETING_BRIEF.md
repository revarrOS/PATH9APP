## Copy Tone Guidelines

### Do ✓
- Speak warmly, like a supportive friend
- Use "you" and "your journey"
- Acknowledge the difficulty of treatment
- Emphasize support, not sales
- Be clear and honest about pricing
- Frame as companion, not tool

### Don't ✗
- Use aggressive sales language
- Create false urgency ("Limited time!")
- Hide pricing or use dark patterns
- Be vague about features
- Oversell or make medical claims
- Use corporate jargon

### Example Good Copy
```
"Less than one copay per month for unlimited support"
"Never walk into an appointment unprepared"
"Your data is always yours"
```

### Example Bad Copy
```
"Premium features unlocked!"
"Don't miss out on this exclusive offer!"
"Advanced enterprise solution"
```
