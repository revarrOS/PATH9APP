## TRIAL & CONVERSION FLOW

### What happens when "Start 30-Day Free Trial" is clicked:

```typescript
// User must be logged in first
if (!user) {
  // Redirect to sign-in with return path
  router.push('/sign-in?returnTo=/settings/pricing');
  return;
}

// Then call the upgrade function
const result = await subscriptionService.upgradeToTrial(
  user.id,
  'plus' // or 'complete'
);

if (result.success) {
  // Show success toast/message
  alert('Trial started! You now have full access to Path9 Plus for 30 days.');

  // Redirect to dashboard or onboarding
  router.push('/(tabs)');
}
```

### NO Payment Processing Yet
Just placeholder buttons that trigger the `upgradeToTrial()` function.
This sets `trial_start_date` and `trial_end_date` in the database.
Actual payment/Stripe integration comes later.

### Database Already Has:
- `subscription_tiers` table with pricing data
- `subscriptionService.getAllTiers()` to fetch tiers
- `subscriptionService.upgradeToTrial(userId, tier)` to start trial
- `subscriptionService.checkAccess(userId, feature)` to verify access
